# 全网小说聚合查询
## 项目简介
我们提供一个聚合查询的网页，为你搜寻全网的小说，让你可以随时查看小说的最新章节

## 项目网址：
### [全网小说聚合查询](https://poly.arics.xyz/index)

## 使用须知
**在使用之前我们非常希望你能阅读并理解我们的免责声明，在每次进入页面时我们都会为您展示。**
